## 🚀 Overview

Submanac is a next-generation prediction market platform designed to incentivize **early conviction**.

Traditional prediction markets reward accuracy — but often overlook *timing*. Submanac flips the model by rewarding traders who take positions **before consensus forms**, when their signals carry the most value.

---

## ✨ Key Features

* ⏱️ **Early Position Rewards**
  Earn more by committing earlier — before the crowd catches on.

* 📊 **Signal-Driven Markets**
  Markets designed to surface meaningful, high-signal insights.

* 🧠 **Conviction-Based Scoring**
  Rewards are based on both correctness *and* timing.

* 🔗 **Onchain Execution**
  Transparent, verifiable, and trustless market mechanics.

* ⚡ **Fast & Intuitive UX**
  Built for both casual traders and power users.

## 🧩 How It Works

1. **Browse Markets**
   Explore active prediction markets across various topics.

2. **Commit Early**
   Take positions before trends emerge.

3. **Hold Conviction**
   Maintain your stance as the market evolves.

4. **Earn Rewards**

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/submanac.git
cd submanac
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Setup environment

```bash
cp .env.example .env
```

Fill in the required environment variables.

### 4. Run development server

```bash
npm run dev
```
